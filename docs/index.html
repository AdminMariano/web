Controlando un GLCD GDM12864-5 con controlador ST7920 usando CCS C y PIC18F25K22

  • Pantalla: GLCD GDM12864-5 (128x64 píxeles, monocromo)
  • Controlador: ST7920 en modo paralelo 8 bits
  • Microcontrolador: PIC18F25K22
  • Lenguaje: CCS C Compiler
  • Driver: Incluye inicialización en modo básico y gráfico
  • Funciones: Dibujar píxeles, rellenar pantalla, actualizar con framebuffer, imprimir texto y gráficos
  • Ventaja: Uso de memoria intermedia (double buffer) para evitar parpadeo y mejorar rendimiento
  • Aplicaciones: Interfaces gráficas simples, menús, instrumentación, logotipos y monitoreo embebido

Este driver probado físicamente permite un control completo del GLCD, integrando gráficos y texto en proyectos embebidos de forma eficiente.
